## 1. Array (🔥 Most Important)

* Two Pointer
* Sliding Window (Fixed & Variable)
* Prefix Sum
* Kadane’s Algorithm
* Subarray Problems
* Merge Intervals
* Binary Search on Array

---

## 2. Dynamic Programming

* 1D DP (Fibonacci, Climbing Stairs)
* 2D DP (Grid Problems)
* Knapsack (0/1, Unbounded)
* Longest Common Subsequence (LCS)
* Longest Increasing Subsequence (LIS)
* DP on Strings
* DP on Stocks

---

## 3. Recursion

* Backtracking
* Subset / Subsequence
* Permutations / Combinations
* Recursion Tree
* Divide & Conquer

---

## 4. String

* Sliding Window on Strings
* Pattern Matching
* Anagram Problems
* Palindrome
* String Hashing (basic)
* KMP (advanced, optional)

---

## 5. Greedy

* Activity Selection
* Interval Scheduling
* Jump Game
* Fractional Knapsack
* Huffman Coding (basic idea)

---

## 6. Stack

* Monotonic Stack
* Next Greater Element
* Valid Parentheses
* Min Stack
* Histogram (Largest Rectangle)

---

## 7. Trees

* DFS (Inorder, Preorder, Postorder)
* BFS (Level Order)
* Binary Search Tree (BST)
* Lowest Common Ancestor
* Tree Diameter
* Balanced Tree
* Path Sum Problems

---

## 8. Graphs

* BFS / DFS
* Topological Sort
* Cycle Detection
* Shortest Path (Dijkstra, BFS)
* Union-Find (Disjoint Set)
* Minimum Spanning Tree (Kruskal, Prim)

---

## 9. Bit Manipulation

* XOR Tricks
* Set / Clear / Toggle Bits
* Power of 2
* Subset Generation using Bits
* Bit Masking (advanced)

---

## 10. Hash Map

* Frequency Count
* Prefix Sum + HashMap
* Two Sum Pattern
* Grouping (Anagrams)
* Caching

---

## 11. Sorting Algorithms

* Quick Sort
* Merge Sort
* Heap Sort
* Counting Sort (basic idea)
* Custom Sorting

---

## 12. Heap (Priority Queue)

* Top K Elements
* Kth Largest/Smallest
* Sliding Window Maximum
* Merge K Sorted Lists
* Median of Stream

---

## 13. Queue / Deque

* Sliding Window Maximum (Deque)
* BFS Queue Usage
* Circular Queue
* Monotonic Deque

---

## 14. Range Structures

* Segment Tree (Range Query)
* Binary Indexed Tree (Fenwick Tree)
* Range Sum Query
* Lazy Propagation (advanced)

---

## 15. Linked List

* Reverse Linked List
* Cycle Detection (Floyd’s)
* Merge Lists
* Middle of List
* LRU Cache (important combo)

---

## 16. Trie

* Insert / Search / Prefix
* Word Search
* Auto-complete
* Bitwise Trie (advanced)

---

## Learning Order

👉 Array → String → HashMap → Stack → Sliding Window → Recursion → Trees → Graphs → DP
