# DSA (Data Structures & Algorithms)

## DSA (Data Structures & Algorithms)

Data Structures and Algorithms (DSA) refer to the concepts used to organize, manage, and process data efficiently. Data structures store data, while algorithms define step-by-step procedures to solve problems. Together, they help optimize time and space complexity in software development.

---

## Array

An array is a linear data structure that stores elements of the same type in contiguous memory locations. It allows fast access using index positions, typically in constant time O(1). Arrays are widely used for storing and processing collections of data efficiently.

---

## String

Strings are sequences of characters used to represent text and are a common data structure in programming. Operations include searching, pattern matching, and manipulation like reversing or substring extraction. Algorithms like KMP and Rabin-Karp are used for efficient string processing.

---

## Recursion

Recursion is a technique where a function calls itself to solve smaller instances of the same problem. It consists of a base case to stop and recursive calls to reduce the problem. It is widely used in tree traversal, backtracking, and divide-and-conquer algorithms.

---

## Stack

A stack is a linear data structure that follows the Last In First Out (LIFO) principle. Operations include push, pop, and peek, all in constant time. It is used in expression evaluation, recursion handling, and undo/redo operations.

---

## Hash Map

A hash map stores key-value pairs and provides average constant time complexity for insertion and lookup. It uses a hash function to map keys to indices. It is widely used for frequency counting, caching, and quick lookups.

---

## Sorting Algorithms

Sorting algorithms arrange data in a specific order, usually ascending or descending. Common algorithms include quicksort, mergesort, and bubblesort, each with different time complexities. Efficient sorting is crucial for optimizing searching and data processing.

---

## Linked List

A linked list is a linear data structure where each node points to the next node in the sequence. It allows dynamic memory allocation and efficient insertions/deletions. Types include singly, doubly, and circular linked lists.

---

## Queue / Deque

A queue follows the First In First Out (FIFO) principle, where elements are added at the rear and removed from the front. A deque (double-ended queue) allows insertion and deletion from both ends. They are used in scheduling, BFS, and sliding window problems.

---

## Dynamic Programming

Dynamic Programming (DP) solves complex problems by breaking them into overlapping subproblems and storing results to avoid recomputation. It uses memoization (top-down) or tabulation (bottom-up) approaches. DP is commonly applied in optimization problems like knapsack, Fibonacci, and shortest paths.

---

## Greedy

Greedy algorithms make the optimal local choice at each step to achieve a global optimum. They are efficient because they don’t reconsider previous decisions. Common examples include activity selection, Huffman coding, and coin change (with certain constraints).

---

## Trees

Trees are hierarchical data structures consisting of nodes connected by edges with a single root. They are used to represent relationships like file systems or organizational data. Common types include binary trees, BSTs, AVL, and heaps.

---

## Graphs

Graphs consist of nodes (vertices) connected by edges and are used to model networks. They can be directed/undirected and weighted/unweighted. Traversal algorithms like BFS and DFS are fundamental for solving graph problems.

---

## Bit Manipulation

Bit manipulation involves performing operations directly on binary representations of numbers. It is used for optimization and solving problems efficiently with low-level operations. Common techniques include AND, OR, XOR, shifting, and masking.

---

## Heap

A heap is a complete binary tree that satisfies the heap property (min-heap or max-heap). It is commonly implemented using arrays for efficiency. Heaps are used in priority queues, scheduling, and algorithms like heapsort.

---

## Range Structures

Range data structures are used to efficiently answer queries over a range of values. Examples include segment trees, Fenwick trees, and sparse tables. They are useful in problems involving range sum, minimum, or updates.

---

## Trie

A trie is a tree-based data structure used for storing strings, especially for prefix-based searching. Each node represents a character, and paths form words. It is commonly used in autocomplete, dictionaries, and search suggestions.
